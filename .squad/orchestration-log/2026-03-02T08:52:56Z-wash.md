## Work Summary

Completed full rework of Radial.lua with OPie-style stick-based interaction model.

### What Was Done

- **Redesigned wheel interaction:** Replaced two-stage trigger+buttons model with intuitive stick-based selection
- **Visual expansion:** Increased wheel radius 120px→200px, icon size 52px→64px for couch visibility
- **New core functions:**
  - `GetStickAngle()` — polls left stick from C_GamePad, calculates angle with dead zone
  - `AngleToSlot()` — maps angle (0-360°) to slot index (1-12) using OPie's formula
  - `UpdateStickSelection()` — OnUpdate polling loop, highlights slot based on stick angle
  - `SetSlotHighlight()` — applies/removes gold border + 1.3x scale to slot frames
  - `OpenWheel()` / `ConfirmAndClose()` — trigger press/release handlers
  - `StartStickPolling()` / `StopStickPolling()` — control polling lifecycle
- **Simplified bindings:** Removed face button bindings from wheel mode (PAD1-4), kept only bumpers (L1/R1) and trigger (RT)
- **Interface wheel execution:** Added `execute()` functions to all INTERFACE_WHEEL_LAYOUTS, direct button clicks for Blizzard UI
- **Combat safety maintained:** All polling read-only, no SetAttribute in OnUpdate, SecureActionButton clicks for user wheels
